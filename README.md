# DTTOT APU PPT - PPATK

Clean & Extract data DTTOT APU PPT from PPATK

## Installing

```node
npm install
```

## Running

#### Format

```node
npm start <input xlsx> <out filename>
```

#### Example

```node
npm start ./202207220950531.xlsx out2
```
